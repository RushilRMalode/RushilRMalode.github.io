Implementation of a scalar autodiff engine (in the spirit of micrograd) that builds a computation graph over scalar values and backpropagates gradients through it. An MLP built entirely on top of this engine is trained to learn the XOR function, a classic non-linearly-separable problem that requires a hidden layer to solve.
